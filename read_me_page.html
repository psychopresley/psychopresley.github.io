Hi, there! This page is supposed to be just a cool way to visualize some aspects of the coronavirus outbreak, so you won't find here any deep analysis, modelling or advanced machine learning estimation of the near future. I will acknowledge here most of the tools that made possible to build this page and also provide some links if you're interested in a more deep analysis of the pandemics.

All graphics presented here were build with Flourish Studio. It's a very interesting web based visualization app with the option of creating storytellers, which is a cool way to present your data. As an example you can check my own storytelling for this project here.

All data is made available by Johns Hopkins University. I use python 3 to collect all the available data from their github repository and build the dataset that provides the numbers presented here. If you're interested in check my work just visit my github repository for this project. Aside from this page I also have a dashboard build with Tableau public. There you will find a deeper analysis of the pandemics with much more visualization options. It's worth to take a look. Also I made available a dataset with the compiled numbers that can be accessed here. It has some saved queries that can be run and provide you with cool informations about the pandemics. Feel free to check it!

Finally, I only have a free acount in the Flourish app. This does not allow me to update all data automatically so I have to do it manually, which means that I will not update the data everyday. With a paid membership account I'd have the possibility of update the graphics using a google sheet API. No problem. So I'll update all graphics twice a week, on sundays and thursdays in the morning. All number presented here were updated on january 21th, 2021, so I see you again next sunday!
